# Security Policy

Sentra CLI scans local AI Agent assets, so vulnerability reports are especially welcome.

## Supported Versions

Security fixes are applied to the latest release line.

## Reporting a Vulnerability

Please do not disclose vulnerabilities publicly before maintainers have had time to investigate.

Include:

- Affected version or commit.
- Operating system and installation method.
- Reproduction steps.
- Impact and any known workaround.

If the project is mirrored to GitHub, use GitHub private vulnerability reporting when available. Otherwise, contact the maintainers through the repository owner.

## Scope

Reports are in scope when they affect Sentra CLI, `sentra-lib`, bundled rule loading, local asset discovery, or scan output handling.
